File Goblin helps easily organize, search with hierarchical tags, and share media. Whether that's images, memes, videos, links, etc.

## Technologies

- SvelteKit
- Better-Auth
- PostgreSQL (NeonDB, DrizzleORM)
- Ubuntu VPS
- Docker
- File storage (Backblaze B2 with Cloudflare CDN)
- shadcn-svelte
